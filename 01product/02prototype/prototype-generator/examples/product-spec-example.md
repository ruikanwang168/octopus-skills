# 示例需求文档：项目管理系统

## 1. 产品背景

这是一个用于企业内部项目过程管理的系统，帮助管理人员查看项目进展、维护项目基本信息、管理项目成员，并跟踪项目状态。

## 2. 用户角色

- 管理员：可以查看、创建、编辑、删除项目，并维护项目成员。
- 普通成员：可以查看自己参与的项目和项目详情。

## 3. 功能模块

### 3.1 工作台

- 展示项目总数、进行中项目数、延期项目数。
- 展示我的待办。
- 支持点击进入项目管理模块。

### 3.2 项目管理

- 支持项目查询。
- 支持新建项目。
- 支持查看项目详情。
- 支持编辑项目信息。
- 支持删除项目。
- 支持维护项目成员。

## 4. 业务流程

用户进入工作台后，可以点击“项目管理”进入项目列表页。  
在项目列表页可以搜索项目、筛选项目状态、新建项目、查看项目详情、编辑项目、删除项目。  
在项目详情页可以查看项目基本信息、项目进度、项目成员，并进行成员管理。

## 5. 字段说明

### 项目列表字段

- 项目名称
- 项目编号
- 项目状态
- 负责人
- 开始日期
- 结束日期

### 新建项目字段

- 项目名称
- 项目编号
- 项目类型
- 负责人
- 开始日期
- 结束日期
- 项目描述
